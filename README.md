# Portfolio

Персональный сайт-портфолио.

## Возможности

- Адаптивная верстка
- Витрина проектов
- Блок навыков
- Смены темы
- i18n-контент (EN/RU)

## Технологии

- HTML
- CSS
- SCSS
- JavaScript

## Структура проекта

- index.html
- index.js
- script/ (логика сайта, например проекты)
- style/ (SCSS и скомпилированный CSS)
- i18n/ (JSON-файлы локализации)

## Быстрый старт

- Откройте index.html в браузере.

## Заметки по разработке

- Строки локализации лежат в i18n/en.json и i18n/ru.json.
- Стили пишутся в style/style.scss и частях.

## Деплой

Разместите папку на любом статическом хостинге (GitHub Pages, Netlify, Vercel и т. д.).
